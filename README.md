# KBOT RVIZ
## Description
This package is using for monitor with rviz
## Installation on tour PC
```
$ cd ~/catkin_ws/src
$ git clone https://github.com/aims-lab-kmitl/kbot_rviz.git
$ cd ~/catkin_ws
$ catkin_make
```
## Command
- Kbot Slam
```
$ roslaunch kbot_rviz slam.launch
```
- Kbot Navigation
```
$ roslaunch kbot_rviz navigation.launchV
```
