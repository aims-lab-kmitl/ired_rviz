# ired RVIZ
## Description
This package is using for monitor with rviz
## Installation on tour PC
```
$ cd ~/catkin_ws/src
$ git clone https://github.com/aims-lab-kmitl/ired_rviz.git
$ cd ~/catkin_ws
$ catkin_make
```
## Command
- ired Slam
```
$ roslaunch ired_rviz slam.launch
```
- ired Navigation
```
$ roslaunch ired_rviz navigation.launchV
```
